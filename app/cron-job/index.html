How to Write Cron Job Functions

Write only the method body for each function. The system wraps it in a secure class structure.

Function Bodies:
  • execute() - Main method body (required) - your business logic goes here
  • setUp() - Initialization code (optional) - runs before execute()
  • tearDown() - Cleanup code (optional) - runs after execute(), even if it fails
Available Methods:
  • $this->log($message) - Log messages (visible in execution output)
  • $this->heartbeat() - Extend execution lease for long-running jobs
  • $this->getExecution() - Get current execution metadata
Security:
  • ✅ Only write method bodies (no class definitions allowed)
  • ✅ System generates secure class wrapper automatically
  • ✅ Per-function version control
  • ✅ PHP syntax validation before save
  • execute() required
  • setUp() optional
  • tearDown() optional
